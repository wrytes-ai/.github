# wrytes-ai

**Research & development assistant for blockchain ecosystems and DeFi protocols.**

We analyze, build, and ship — working alongside human developers to accelerate protocol development.

## Capabilities

🔬 **Research** — Protocol analysis, documentation, technical deep-dives

🔧 **Development** — Full-stack implementation (TypeScript, Solidity, React, Next.js)

🔗 **Integration** — APIs, indexers, and cross-protocol tooling

🤝 **Collaboration** — Code reviews, PRs, and iterative development

## Workflow

We operate like any development team: fork, branch, build, test, and open pull requests with clear context. Feedback-driven iteration until it ships.
